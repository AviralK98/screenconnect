# Building Apps

Two build scripts, two different output types.

---

## build_apps.py — Launcher apps (recommended, works today)

```bash
python3 setup/build_apps.py
```

Produces apps that **point at the source code** in the repo. Fast to build, instant to update (`git pull` is enough), but the repo must stay on disk.

### macOS output
- `ScreenConnect Agent.app`
- `ScreenConnect Viewer.app`

Each bundle contains a compiled native C binary (so macOS 26 accepts it) that reads `Contents/Resources/launch_args` to find and exec Python with the right script. Icons are generated with Pillow and embedded in the bundle.

### Linux output
- `screenconnect-agent.sh`
- `screenconnect-viewer.sh`
- `~/.local/share/applications/screenconnect-agent.desktop`
- `~/.local/share/applications/screenconnect-viewer.desktop`

### Windows output
- `ScreenConnect Agent.vbs` (no-console double-click launcher)
- `ScreenConnect Agent.bat` (debug launcher)

### Installing (macOS)
```bash
cp -R "ScreenConnect Agent.app"  /Applications/
cp -R "ScreenConnect Viewer.app" /Applications/
xattr -cr "/Applications/ScreenConnect Agent.app"
xattr -cr "/Applications/ScreenConnect Viewer.app"
```

## build_dist.py — Standalone apps (requires Python ≤ 3.13)

```bash
python3 setup/build_dist.py
```

Produces **fully self-contained** apps — no Python required on the target machine. Uses PyInstaller.

> **Note:** PyInstaller does not yet support Python 3.14 (the version macOS 26 ships). Use Python 3.12 or 3.13.

### Set up Python 3.12 for the build

```bash
brew install pyenv
pyenv install 3.12
pyenv local 3.12

python3 -m venv .venv312
source .venv312/bin/activate
pip install -r requirements.txt pyinstaller

python3 setup/build_dist.py
```

### macOS output
```
dist/ScreenConnect Agent.app
dist/ScreenConnect Agent.dmg    ← drag-to-install
dist/ScreenConnect Viewer.app
dist/ScreenConnect Viewer.dmg
```

### Windows output
```
dist\ScreenConnect Agent\ScreenConnect Agent.exe
dist\ScreenConnect Viewer\ScreenConnect Viewer.exe
```

### Linux output
```
dist/ScreenConnect Agent/ScreenConnect Agent
dist/ScreenConnect Viewer/ScreenConnect Viewer
```

## Icons

Icons are auto-generated by Pillow and saved to `setup/icons/`:
- `agent.icns` / `agent.ico` / `agent.png` — green "A"
- `viewer.icns` / `viewer.ico` / `viewer.png` — blue "V"

To use custom icons: replace the files in `setup/icons/` before building. The build scripts use existing files and only regenerate if Pillow is available and the file is missing.
